크롤링 아이디어

1. 목표 주소까지 접근
목표 데이터가 있는 세부 도메인으로 접근하는 것이 불필요한 코드를 줄일 수 있음

2. 팝업창 관련 문제
웹 페이지 접근 시 팝업창이 뜨는 경우, 이를 끄기 위한 작업을 해주어야 함.
특정 조건에 따라서 팝업창 발생 여부가 달라지기 때문에 그와 관해서도 처리가 필요함

3. 반복 작업과 규칙 찾기
매번 모든 element를 찾아서 처리할 수는 없기 때문에,
list로 되어있는 element들은 규칙을 찾아서 for문으로 처리해야 함

4. 번외
* 클릭을 진행하면 새창이 열려 창이 변경됨 -> 창 변경 메서드를 활용
* 로그인 진행시 캡차 문제가 발생 -> 사이트별로 캡차 우회 방법이 존재
* 원하는 데이터가 팝업창으로 뜸 -> 셀레늄만으로 불가능(심화편)
* 데이터를 .text로 긁었지만 긁히지 않음 -> time.sleep()으로 기다림
